Utvärdering av Milligram

Om man snabbt vill skapa en stylad webbplats är Milligram ett CSS-ramverk som enkelt ger en god grundCSS att stå på, på ett lättanvänligt sätt. Man behöver inte använda sig av klasser på alla element utan Milligram känner igen de element som används och ger CSS-kod utifrån det. Det finns specifika klasser att lägga in för exempelvis knappar, men det är få valmöjligheter med endast ett par-tre stycken, vilket gör det lätt att bestämma sig för hur man vill att det ska se ut. Här nedan finns exempel på en tabell samt ett formulär där jag endast använt mig av Milligram för att styla, inget annat. Som kan ses är Milligramstilen väldigt basic, men det räcker gott för dem som snabbt vill kunna styla en webbplats effektivt utan att behöva lägga till alltför många klasser till elementen. Små saker behöver läggas till som exempelvis padding/margin mellan vissa delar i formuläret och så vidare men generellt sett är det väldigt simpla saker att lägga till för att få det att se snyggt ut. Under tiden som jag har läst om och prövat Milligram upplever jag att det är väldigt enkelt för nybörjare av ramverk att använda, samt spar in en hel del tid när man slipper att lägga in klasser för varje element. gentemot exempelvis Bootstrap där man lägger in klasser för varje element. Vill man däremot ha mer avancerad CSS är inte Milligram rätt verktyg utan då bör man välja något annat CSS-ramverk som till exempel Tailwind eller Bootstrap.
